# MicroLab – AI Lab Assistant for Microbiology

Bringing intelligent diagnostics, SOP guidance, and knowledge synthesis to 
labs in East Africa and other resource-constrained settings.

## Getting Started
```bash
git clone https://github.com/jbo4k/microlab.git
cd microlab
pip install -r requirements.txt
python scripts/setup_env.sh
```

## Key Features (MVP)
- SOP library (searchable, auto-retrieval)
- Biochemical test interpreter (results → differentials)
- Experiment logger (ELN)
- Q&A troubleshooting (RAG + LLM)
